# متجر من أجلك للهدايا (For You Gifts)

![For You Gifts](public/favicon.png)

منصة الكترونية (كتالوج) لمتجر "من أجلك" للهدايا. تم بناء المشروع باستخدام لغة PHP الأصيلة (Vanilla PHP) بتصميم هيكلي مخصص يشبه (MVC) لضمان سهولة التوسع، الأمان، وسهولة الصيانة، مع واجهة لوحة تحكم (Admin Panel) متكاملة لإدارة المتجر.

## الميزات الرئيسية (Key Features)

* **هيكلة برمجية نظيفة (Custom Architecture):** يعتمد المشروع على فصل المنطق (Backend Handlers) عن العرض (Views/Pages) للحفاظ على كود نظيف وقابل للقراءة.
* **نظام التوجيه (Routing System):** نظام توجيه مركزي (`index.php`) يقوم بإدارة جميع الطلبات والروابط (Clean URLs) مما يحسن من تقنيات تحسين محركات البحث (SEO).
* **لوحة تحكم متكاملة (Admin Dashboard):** واجهة إدارة آمنة تتيح للمدراء إدارة (الفئات، الأقسام الفرعية، المنتجات، صور المنتجات، وإعدادات الموقع).
* **إدارة المنتجات الديناميكية:** إمكانية إضافة منتجات مع دعم للصور المتعددة (صور رئيسية وفرعية) لتوضيح تفاصيل الهدايا.
* **الأمان (Security):** 
  * تشفير كلمات المرور باستخدام خوارزميات التشفير القوية في PHP.
  * حماية المجلدات الحساسة وملفات الإعدادات عبر `.htaccess`.
* **قاعدة بيانات علائقية:** تصميم أمثل لجداول قاعدة البيانات (MySQL) باستخدام مفاتيح أجنبية (Foreign Keys) وفهرسة (Indexing) متقدمة لضمان سرعة الاستعلامات وتكامل البيانات.

## التقنيات المستخدمة (Tech Stack)

* **الواجهة الخلفية (Backend):** PHP (Vanilla)
* **قاعدة البيانات (Database):** MySQL (PDO)
* **الواجهة الأمامية (Frontend):** HTML5, CSS3, JavaScript
* **الخادم (Server):** Apache (XAMPP/MAMP for local development)

## هيكل المشروع (Project Structure)

```text
/
├── app/                # المنطق الخلفي وإعدادات قاعدة البيانات
├── pages/              # صفحات العرض العامة (الرئيسية، المنتجات، اتصل بنا...)
├── public/             # الأصول الثابتة (CSS, JS, Images)
├── templates/          # القوالب ومكونات العرض (Header, Footer)
├── admin/              # لوحة التحكم المستقلة
├── sql/                # ملفات قاعدة البيانات (schema.sql)
├── uploads/            # مسار رفع ملفات وصور المستخدمين والمنتجات
└── index.php           # الموجّه الرئيسي للطلبات (Main Router)
```

## متطلبات التشغيل (Prerequisites)

* خادم ويب (مثل Apache مدعوم بـ XAMPP أو WAMP).
* إصدار PHP 7.4 أو أحدث.
* قاعدة بيانات MySQL/MariaDB.

## التثبيت والإعداد المحلي (Installation & Setup)

1. **استنساخ المستودع (Clone Repository):**
   ```bash
   git clone https://github.com/yourusername/foryou-gifts.git
   ```

2. **نقل ملفات المشروع:**
   قم بنقل مجلد المشروع إلى المجلد الجذري لخادم الويب الخاص بك (مثلاً `htdocs` في XAMPP أو `www` في WAMP).

3. **إعداد قاعدة البيانات:**
   * افتح أداة إدارة قواعد البيانات (مثل phpMyAdmin).
   * قم بإنشاء قاعدة بيانات جديدة (مثلاً `foryou_db`).
   * قم باستيراد الملف الموجود في مسار `sql/schema.sql` إلى قاعدة البيانات التي أنشأتها للتو. (يحتوي الملف على جداول المستخدمين، وتصنيفات هدايا وبيانات تجريبية جاهزة).

4. **إعداد الاتصال بقاعدة البيانات:**
   * اذهب إلى الملف `app/config/database.php` (إذا كان موجوداً) أو قم بإنشائه إذا كان يعتمد على `app/config/site.php`.
   * قم بتعديل بيانات الاتصال (اسم قاعدة البيانات، اسم المستخدم `root`، وكلمة المرور) لتتطابق مع إعداداتك المحلية.

5. **تشغيل المشروع:**
   * تأكد من أن خوادم Apache و MySQL تعمل.
   * افتح المتصفح وانتقل إلى الرابط المحلي: `http://localhost/foryou-gifts`
